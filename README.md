 Advanced CPU Architecture & Hardware Accelerators - VHDL Labs

Welcome to my hardware development repository. This project covers the end-to-end design process of digital systems and processors, from the physical layer to the application layer, implemented in VHDL and validated on FPGA.

## Project Overview
These labs focus on professional hardware design methodologies, emphasizing the full development cycle:
* **Design & Optimization:** Combinational, Synchronous, and Integrated digital systems designed under **PPA (Power, Performance, and Area)** constraints.
* **Verification:** Functional verification, Timing verification, and hardware validation on **FPGA** chips.
* **Computer Architecture:** Deep dive into ISA and Microarchitecture, with a strict separation between **Control Unit** and **Datapath**.

## Development Milestones

## Phase 1: Digital Logic Fundamentals
Implementation of foundational digital systems using VHDL, focusing on synthesis-friendly coding styles and comprehensive Testbench verification.

## Phase 2: CPU Core Development
* **Multicycle Processor:** Designing a separate Control and Datapath architecture.
* **Single-Cycle Processor:** Full implementation and FPGA deployment.
* **V-RISC Pipelined Core:** Development of an N-stage single-threaded pipelined RISC-V processor.

### Phase 3: Final Project - Full MCU System
Design and integration of a complete **Microcontroller Unit (MCU)** on an FPGA, based on a pipelined core. The system features:
* **Peripheral Modules** (IO, Timers, etc.).
* **Interrupt Controller.**
* **Hardware Accelerators (HW Accelerators)** – optimized from the physical layer up to the application software.
